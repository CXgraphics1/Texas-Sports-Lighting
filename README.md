
# Texas Sports Lighting — Static Website

This is a complete, multi‑page static site scaffold for **Texas Sports Lighting** with HTML, CSS, and JS.

## Pages
- Home (`/index.html`)
- About (`/about.html`)
- Services (`/services/*.html`): Tennis Centers, Football Stadium, Indoor Arenas, Aquatic Centers, Sports Complex
- Products (`/products.html`)
- Resources (`/resources/*`): Gallery, Blog
- Retrofits & New Build (`/retrofit-new-build.html`)
- Controls (`/controls.html`)
- Contact / Quote (`/contact.html`)

## How to use
1. Replace placeholder copy with your **own original text**. Do not copy third‑party site text verbatim.
2. Swap the placeholder images in `/assets/img/` with your project photos.
3. Update contact info and `form` action URLs.

## Dev notes
- Fonts: Inter (Google Fonts)
- No frameworks; vanilla HTML/CSS/JS
- Responsive header, dropdowns, lightbox, and basic form validation
